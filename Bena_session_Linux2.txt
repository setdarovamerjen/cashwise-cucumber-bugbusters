find . -name "*.feature" 2>/dev/null - finds only existed files whatever that has .feature
find . -name "*test*" 2>/dev/null = finds that contains the word 'test' and with existed files
find . -iname "POM>XML" ==> FOR CASE INSENSITIVE RESULTS


grep -i 'something' report.log ==> same thing but case insensitive

grep -n 'Error' report.txt ==> returns the strings with numbers

grep -c 'Error' report.txt ==> return sthe count of String apprearances

grep 'SearchString' directoryPath ==> returns the matching string from all file inside the directory

NOTE: both grep and find commands are not recommended to be used to search for strings in binary, non text files: PDF, Word, images
